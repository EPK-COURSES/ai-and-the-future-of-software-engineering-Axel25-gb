The Impact of AI on Developer Productivity: Evidence from GitHub Copilot

Authors: Sida Peng, Eirini Kalliamvakou, Peter Cihon, Mert Demirer  |  Published: February 2023  |  Publisher: Microsoft Research / arXiv
AI Productivity GitHub Copilot Controlled Experiment Software Engineering

Abstract

Generative AI tools hold promise to increase human productivity. This paper presents results from a controlled experiment with GitHub Copilot, an AI pair programmer. Recruited software developers were asked to implement an HTTP server in JavaScript as quickly as possible. The treatment group, with access to the AI pair programmer, completed the task 55.8% faster than the control group. Observed heterogeneous effects show promise for AI pair programmers to help people transition into software development careers.

Key Findings

  • Developers with GitHub Copilot access completed the assigned task 55.8% faster than those without it.
  • The experiment was conducted as a randomized controlled trial with recruited software developers.
  • The task involved implementing an HTTP server in JavaScript — a realistic but self-contained coding task.
  • Results suggest AI pair programmers may be especially useful for helping new developers transition into software development careers.
  • Heterogeneous effects were observed — the benefit varied depending on the experience level and background of the developer.

Why This Matters

This paper is one of the most widely cited pieces of evidence for AI-driven productivity gains in software engineering. While the task is relatively contained (implementing a single server), the controlled design and significant effect size make it a strong baseline reference. It is frequently cited by companies (including Shopify and Duolingo) when justifying AI-first strategies.

Importantly, subsequent studies on more complex tasks (e.g., Google's 2024 internal RCT on enterprise-grade coding work) found more modest gains (~21%), suggesting the simpler the task and less experienced the developer, the larger the AI productivity benefit tends to be.
